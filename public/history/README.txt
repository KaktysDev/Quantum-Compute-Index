  google.png    →  the Google quantum computer photo  (timeline point 2)

To replace one, keep the same filename. If you switch format (.jpg/.webp),
update the matching `src` in src/app/history/page.tsx.

Images are auto-converted to monochrome (grayscale) on the page to match the design.
After adding/replacing images, rebuild (npm run build) so the server serves them.
